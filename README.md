# ❤️ Heart Disease Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project is completed as part of the **CodSoft Data Analytics Internship - Task 2**.

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Heart Disease Dataset to understand the data, identify patterns, detect relationships between variables, and gain meaningful insights through data visualization.

---

## 📂 Dataset
- **Dataset:** Heart Disease Dataset
- **File:** heart.csv

---

## 🛠️ Tools & Libraries Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Exploratory Data Analysis Performed

- Loaded the dataset
- Explored dataset structure
- Checked dataset dimensions
- Displayed column names
- Generated descriptive statistics
- Checked missing values
- Checked duplicate records
- Created meaningful visualizations
- Analyzed relationships between variables
- Derived insights from each visualization

---

## 📈 Visualizations

- Age Distribution
- Heart Disease Distribution
- Distribution of Cholesterol Levels
- Resting Blood Pressure Distribution
- Maximum Heart Rate Distribution
- Chest Pain Type vs Heart Disease
- Gender vs Heart Disease
- Age vs Cholesterol (Scatter Plot)
- Correlation Heatmap

---

## 🔍 Key Insights

- Most patients are between 50 and 60 years of age.
- Heart disease cases are slightly higher than non-heart disease cases.
- Most patients have normal to moderately high resting blood pressure.
- Maximum heart rate is concentrated between 140 and 170 bpm.
- Chest pain type shows a noticeable relationship with heart disease.
- Male patients have more heart disease cases than female patients.
- Cholesterol values contain a few outliers.
- The correlation heatmap highlights relationships among important medical features.

---

## ✅ Conclusion

The Exploratory Data Analysis of the Heart Disease Dataset helped identify important trends, distributions, and relationships among different medical features. The visualizations provide a better understanding of factors associated with heart disease and form a strong foundation for future predictive modeling.
